# Malvin C XMD - WhatsApp Multi-Device Bot 🇿🇼

Welcome to the **v2.0** of Malvin C XMD, now rebuilt with **ES Modules (import)** and **107 mature commands**.

## 🚀 Features
- **Modern Syntax**: Built using `import` for better performance and structure.
- **107 Mature Commands**: Organized for utility, media, and group management.
- **Vercel Optimized**: Ready for serverless deployment.
- **Pairing UI**: Beautiful web interface for easy linking.

## 🛠 Installation

### 1. Local Setup
```bash
# Extract the ZIP
cd malvin-c-xmd
npm install
npm start
```

### 2. Vercel Deployment
1. Upload these files to a GitHub repository.
2. Link the repository to **Vercel**.
3. Deploy. The `vercel.json` handles the routing.

### 3. Pairing Instructions
1. Navigate to your deployed URL (or `localhost:3000`).
2. Enter your phone number in international format (e.g., `2637xxxxxxxx`).
3. Follow the instructions on the screen to get your pairing code.
4. In WhatsApp: **Linked Devices** > **Link a Device** > **Link with phone number instead**.
